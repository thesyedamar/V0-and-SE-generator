# Voiceover and Sound Effects Generator

This project is a lightweight Flask application for generating and saving voiceovers and sound effects through a simple web interface.

## Highlights

- Python backend with a small HTML interface
- Form-based workflow for creating audio assets
- Simple structure that is easy to run, review, and extend
- Useful for content creators, video workflows, and quick audio production tasks

## Tech Stack

- Python
- Flask
- HTML templates

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Then open the local URL shown in your terminal.

## Project Focus

This project demonstrates practical automation: taking a repetitive media task and turning it into a simple tool that can generate usable assets faster.
